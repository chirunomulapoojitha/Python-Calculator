# 🧮 Smart Python Calculator

A modern GUI calculator built using Python Tkinter.

## 🚀 Features
- Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Square & Square Root operations
- Clear and Delete options
- Attractive UI design

## 🛠️ Technologies Used
- Python
- Tkinter GUI Library

## 🎯 Purpose
This project was created as part of my learning journey in Python and GUI development.
